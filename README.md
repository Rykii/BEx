# BEx - FICC业务探查智能体主控

[BEx](https://github.com/Rykii/BEx) 是 FICC（固定收益、货币、商品）业务探查系统的**主控智能体（Orchestrator）**。

## 功能

为金融科技领域资产管理系统产品经理提供以下资产类型的业务储备研究：

- **固定收益类**：债券、收益凭证、债券回购
- **衍生品类**：利率互换(IRS)、收益互换(TRS)、场外期权、CDS、商品期货、权益衍生品
- **外汇类**：NDF、即期外汇、汇率掉期
- **商品类**：碳配额/碳信用、大宗商品
- **其他**：结构化产品、基金、信贷资产

## 工作流程

1. **Recon Agent** - 生成权威信源清单（L1监管机构 > L2交易所 > L3行业协会 > L4法律法规库）
2. **Archivist Agent** - 爬取官方文件、清洗结构化信息、入库Markdown
3. **Analyst Agent** - 生成业务探查文档

## 输出格式

所有生成文档遵循Obsidian兼容格式，包含：
- YAML Front Matter
- 政策法规概览
- 交易所规则解读
- 关键定义与术语
- 风险控制要求
- 参考资料索引
